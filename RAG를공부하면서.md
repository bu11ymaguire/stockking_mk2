# 버핏 투자 분석 RAG 시스템 회고록

## 프로젝트 개요

이 프로젝트는 워렌 버핏의 투자 철학을 RAG(Retrieval-Augmented Generation) 시스템으로 구현하여, 사용자가 특정 기업에 대해 질문하면 버핏의 관점에서 투자 분석을 제공하는 시스템이다.

### 시스템 아키텍처

```
사용자 질문 
  → Perplexity API (시장 데이터 수집)
  → RAG 검색 (버핏 인사이트 추출)
  → OpenAI GPT-4o (종합 분석 생성)
  → 5가지 기준 평가 출력
```

---

## RAG 시스템의 작동 원리

RAG는 **Retrieval(검색) + Augmented(강화된) + Generation(생성)**의 약자로, AI 모델이 답변을 생성하기 전에 관련 문서를 먼저 검색해서 참고하는 방식이다.

### 기본 작동 흐름

1. **사용자 질문 입력**: "테슬라는 좋은 투자처인가?"
2. **질문 벡터화**: 질문을 숫자 배열(임베딩)로 변환
3. **유사도 검색**: 벡터 DB에서 질문과 가장 유사한 버핏의 발언/서한 검색
4. **컨텍스트 구성**: 검색된 문서 + 사용자 질문을 하나로 묶음
5. **LLM 생성**: GPT-4o가 이 컨텍스트를 바탕으로 답변 생성

### 이 프로젝트에서의 구현

- **Perplexity API**: 실시간 시장 데이터를 수집
- **RAG 검색**: 버크셔해서웨이 최신 서한에서 인사이트 추출
- **GPT-4o**: 시장 데이터 + 버핏 철학을 결합하여 5가지 기준(경제적 해자, 경영진 자질, 재무 건전성, 가격 적정성, 이해 가능성)으로 평가

## 프로젝트를 통해 깨달은 것

### AI의 '이해'에 대한 철학적 고찰

이 프로젝트를 하면서 RAG를 통해 얻은 데이터를 vector DB에 넣는 매커니즘에 대해 공부하다가 문득 이런 생각이 들었다.

> "AI는 유저의 입력을 숫자로 변환해서 유사도를 비교하고, 확률통계론적 방법으로 출력을 내기 때문에 '실제로 이해하는 것이 아니다'라는 비판이 있다. 그런데 인간의 뇌도 전기 신호로 작동하는데, 연속적인 현실 세계를 정말 '이해'하는 걸까?"

#### 푸념이 아닌 본질적 질문

이 질문은 의식(consciousness)과 이해(understanding)의 본질에 대한 철학적 문제다. 실제로 철학자들과 인지과학자들이 수십 년간 논쟁해온 주제이기도 하다.

**핵심 논점**:
- **구현 매체의 차별성**: 왜 생물학적 뉴런의 전기신호는 "진정한 이해"이고, 인공 뉴런의 수치 연산은 "단순한 패턴 매칭"인가?
- **연속성의 환상**: 인간의 뇌도 결국 이산적(discrete)인 뉴런 발화로 작동한다. 우리가 느끼는 "연속적인 경험"도 뇌가 만들어낸 통합된 표상일 수 있다.
- **"이해"의 정의 문제**: 만약 "이해"를 기능적으로 정의한다면(적절한 입력에 적절한 출력, 맥락 파악, 추론), AI 모델들도 점점 더 이 기준을 충족하고 있다.

#### 반론도 존재한다

- **주관적 경험(Qualia)**: 인간은 "빨강"을 볼 때 단순히 특정 파장을 처리하는 것 이상의 주관적 경험을 한다. 현재 AI에게 이런 현상적 의식이 있는지는 불명확하다.
- **체화된 인지(Embodied Cognition)**: 인간의 이해는 신체, 감각, 세계와의 상호작용을 통해 형성된다. 텍스트만으로 학습한 모델과는 근본적으로 다를 수 있다.
- **인과적 모델 vs 상관관계**: 일부 연구자들은 인간이 세계에 대한 인과 모델을 구축하지만, 현재 언어모델은 주로 통계적 상관관계에 의존한다고 주장한다.

#### 결론

"진짜 이해"와 "단순한 패턴 매칭"을 명확히 구분하는 것은 생각보다 훨씬 어렵다. 어쩌면 이 구분 자체가 이분법적 오류일 수도 있고, "이해"는 스펙트럼상에 존재하는 것일 수도 있다.

## 마치며

이 프로젝트는 단순히 "버핏처럼 투자하는 AI"를 만드는 것을 넘어, AI가 지식을 다루는 방식에 대한 깊은 고민을 하게 만들었다. 

기술의 표면을 넘어 본질을 고민하는 과정에서, AI의 "이해"와 인간의 "이해"가 어쩌면 생각보다 가까울 수도 있다는 생각이 들었다. 둘 다 결국 패턴을 학습하고, 추상화하고, 맥락 속에서 의미를 구성하는 과정이기 때문이다.
